# Kundeteamet
Kundeteamet er opprettet for å sikre effektiv og brukervennlig support/onboarding til brukere/kunder i henhold til avtalte nivåer og retningslinjer.

## Ansvarsområder
_Ansvarsområdene under skal følges opp med kunde-perspektiv av teamet:_
- Hendelsesprosess på tvers av alle team
- Onboarding av tjenesteeiere
- Cloud Governance
- Teknisk brukerstøtte
- Avtaleforvaltning
- API-Management
- Tjenestenivåstyring (SLA)

## Kontaktperson
Ved spørsmål, kontakt team lead Torkel Buarøy eller den/de som er assignet i teamets Kanban-board.
